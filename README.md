 🚀 Newbert

AI-powered Placement Intelligence Platform for Tier-2 & Tier-3 India

> Stop guessing. See exactly what got your seniors placed.

---

📌 Problem

India has 40M+ college students, and most of them are preparing for placements blindly:

- 😵 No placement clarity  
- 🎓 No alumni access  
- 📚 Wrong courses  
- ⚠️ Fake internships  
- 🎯 Skill mismatch  
- 🗺️ No roadmap  

Students don’t know:
- What skills are required  
- What seniors actually did  
- Why they are getting rejected  

---

💡 Solution

Newbert = Placement Readiness Engine**

Instead of showing random jobs, Newbert tells:
- Where you stand  
- What you are missing  
- What to do next  

---
🔥 Features

 🏆 Alumni Wall
- Real placement + GATE data from your college
- Shows DSA count, projects, skills, package

 📊 Gap Analysis
- Compare with placed seniors  
- Output example:
```

You are 40% ready
Missing: 160 DSA + 1 project + 1 internship

```

 🤖 Placement DNA AI
- Predicts placement probability  
- Example:
```

TCS Digital → 62%
Wipro → 78%

```

 💼 AI Job Matching
- Shows skill match % before applying  
- Categories:
- Apply now  
- Learn & apply  

 📄 Resume AI
- Upload resume + paste job description  
- AI:
- Rewrites resume  
- Improves ATS score  
- Predicts interview questions  

🥇 Leaderboard
- Based on DSA, GitHub, streak  
- College-wise ranking  

 📚 Course Recommendation
- Based on placed students  
- AI suggests best course  

 👨‍🏫 Mentorship
- Watch mentor intro video  
- Book session  

 🗺️ Roadmap
- Personalized placement plan  

---
